# Prithvi Sasikumar Portfolio

Interactive AI engineering portfolio built with React, TypeScript, Motion, and a custom canvas animation system.

**Live site:** https://prithvi-01.github.io/portfolio/

## Run locally

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```

The generated production site is written to `dist/` and can be deployed to Vercel, Netlify, GitHub Pages, or another static host.
